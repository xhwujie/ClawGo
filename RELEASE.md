## v1.0.1

### 🔧 网关服务

- 修复安装向导启动的前台 gateway 导致主界面重启失败的问题
- 重启时自动检测端口占用并强制清理残留进程
- 启动前自动执行 `gateway install` 确保 launchd 服务注册
- 全部逻辑兼容 macOS + Windows 双平台

### 📱 消息平台

- 企业微信、钉钉改为插件安装模式（与 QQ 一致），支持一键安装
- 支持多个插件并发安装，安装后自动验证是否成功
- 未安装插件时禁止写入配置，避免 gateway 启动报错
- Telegram 配置新增「私聊策略」「群聊策略」「流式输出」下拉选项
- 飞书配置新增「私聊策略」「群聊策略」下拉选项
- Discord、Slack 配置新增「私聊策略」「群聊策略」下拉选项
- 选择开放模式时自动设置 `allowFrom=["*"]`，无需手动编辑配置文件
- 企业微信、钉钉、QQ 隐藏「授权配对」按钮（这些平台不需要配对流程）

### 🧭 安装向导

- 「继续配对授权」按钮显示对应频道名称（如「继续飞书配对授权」）
- 「查看教程」改为红色文字链接样式

### 💬 聊天

- 进入聊天页面自动检测连接状态，未连接时自动重试连接（最多 5 次）
